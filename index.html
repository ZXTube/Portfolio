Web Development

YouSheild

YouSheild is a chrome extension that stops the user from falling in youtube's rabbit hole.

It hides all of the video recommendations in the homescreen and the videos that appear on the side when watching a video, it only leaves the search bar. And if a youtube shorts video shows up in the search result it only lets the user watch that one video and doesn't allow him to scroll down for more videos

KalemGPT

KalemGPT is a SaaS tool designed for users in Egypt which chatgpt doesn't allow anyone there to make an account.

It uses Open AI's API to ask chatgpt 3.5 questions and forward the answers to the users

Note: I was planning to keep the first year on free trials for the API as the website slowly becomes popular, but later realized that that strategy doesn't work for startups, You can find the website at kalemgpt.com

BizWebless

BizWebless is a SaaS tool made for web designers who want to find clients.

It uses yelp api to find businesses, then it checks which businesses have a website and which don't, which businesses have a good SEO score or good performance. which businesses have the most rating or number of ratings, or both.
The user can choose to sort businesses from high to low or low to high

This website also has stripe implemented into it to allow users to enter a monthly subscription

20-20-20 is a program made for eye care, it applies the 20-20-20 rule, every 20 minutes, look for 20 seconds, at something 20 feet away
